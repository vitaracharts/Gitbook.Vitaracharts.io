# Transparency Slider

### Controlling Color Opacity with the Transparency Slider :&#x20;

In Version 5.1, a new slider has been introduced that allows users to adjust the transparency of a specific color applied to regions where color opacity can be customized as required. This feature is available in the Thresholds, Appearance, Subtotals, Format, Tooltip, and Grand Totals tabs.

Below is a sample GIF demonstrating how to use the transparency slider. By adjusting the transparency, a value in the chart can be hidden.<br>
